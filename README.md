### This is a set of 5 mini-games in a form of a project called "Brain Games".

#### Requirements:
* Unix-like system
* Node.js v13+

### Commands
#### Installation:
```
git clone https://github.com/YadaDelta/frontend-project-44
cd frontend-project-44
make install
npm link
```

#### Games:
```
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
